# Personal CV Website

A simple and elegant personal CV/resume website built using **HTML** and **CSS**.  
This project showcases my professional background, skills, experience, and contact information in a clean and responsive design.

---

##  Features

- Responsive design
- Clean, modern, and easy-to-read layout  
- Sections for profile, education, skills, and work experience  
- Simple customization — just edit the HTML and CSS files  
- Lightweight and fast (no frameworks or JavaScript required)

---

## Technologies Used

- **HTML5** – for structure and content  
- **CSS3** – for styling and layout
